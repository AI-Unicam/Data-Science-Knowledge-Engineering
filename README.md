# Data Science & Knowledge Engineering

AI methods for complex data analysis, knowledge representation, intelligent reasoning, knowledge graphs, semantic technologies, information retrieval, and decision-support systems.

> **Contributing (University of Camerino collaborators)**
>
> If you are a University of Camerino researcher working in this area, you are welcome to showcase your work here. Rather than copying code into this repository, please add your project as a Git submodule so it remains maintained in its original repository while being discoverable through this collection.
>
> ```bash
> git submodule add https://github.com/<username>/<repository>.git <repository>
> git commit -m "Add <repository> as submodule"
> git push
> ```
>
> Then open a Pull Request with a brief description of the project. This keeps ownership and development in your own repository while making it part of the AI@Unicam research collection.
>
